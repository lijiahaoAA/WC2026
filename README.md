## 🏃 跑起来！(本地部署指南)

*前提条件：请确保你的电脑上装了 Python 3.10+、Node.js 和 PostgreSQL。*

### 1. 搞定后端 (API & 数据库)
```bash
cd backend
# 准备一个 .env 文件，填入你的秘密
echo "DB_PASSWORD=你的数据库密码" > .env
echo "ALIYUN_API_KEY=你的阿里云大模型Key" >> .env

# 安装依赖
pip install fastapi uvicorn psycopg2-binary pydantic openai python-dotenv

# 启动引擎 (端口 10086，吉利)
python main.py
```

### 2. 启动前端 (炫酷大屏)
```bash
cd frontend
npm install
npm run dev
# 然后打开浏览器访问 http://localhost:10087/
```
